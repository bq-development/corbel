## Build and Run

### Dependecies

* [Maven](http://maven.apache.org)
* [Docker Compose](https://docs.docker.com/compose/)

### Build and create Docker images

```
mvn install -DdockerBuild=true
```

### Run with Docker Compose

```
docker-compose up
```
